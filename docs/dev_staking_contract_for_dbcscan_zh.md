deeplink短租竞赛质押合约接口文档(dbcscan)
================

## 描述
deeplink短租竞赛质押合约是用于管理 NFT 质押的智能合约。它提供了多种功能，包括质押、解质押、领取奖励等。

## 全局变量
- 'dailyRewardAmount' uint256: 每日总的奖励数量

- 'totalStakingGpuCount' uint256 : 当前处于质押状态GPU的总数量
